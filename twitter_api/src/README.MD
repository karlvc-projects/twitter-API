# Twitter wrapper  

## Description  

Flask application which will pull user profile information:-  

### Endpoint  
/twitter/profile/\<username\>

### Response  
```
{
  "data": {
    "description": "", 
    "id": "2168773878", 
    "name": "First Last", 
    "profile_image_url": "https://pbs.twimg.com/profile_images/1378752367875729669/XtEIxXjT_normal.jpg", 
    "username": "First38367036"
  }
}
```
